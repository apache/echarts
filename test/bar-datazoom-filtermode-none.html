Bar Chart with DataZoom FilterMode 'none' - Issue #19666

Issue Description:

When using a bar chart with dataZoom filterMode='none' and zooming in to show a single bar, the bar graphic displays an incorrect value that doesn't match the tooltip value.

Expected: The bar height should accurately represent the actual value shown in the tooltip.

Before Fix: The bar height shows a random/incorrect value when zoomed to a single category.

Test Instructions:

  1. Use the slider below the chart to zoom in
  2. Zoom to the maximum level so only a single bar is visible
  3. Hover over the bar to see the tooltip
  4. Verify that the bar height matches the value shown in the tooltip
  5. Try different single bars by adjusting the slider
